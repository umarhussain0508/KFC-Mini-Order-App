# 🍗 KFC Mini Order App

A simple Python console application that allows customers to place orders from a KFC-style menu and receive an instant bill.

## Features

* Order multiple items at once
* Automatic bill calculation
* Dine-In or Takeaway option
* Easy-to-use menu system
* Beginner-friendly Python project

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python main.py
```

## Example

```text
Enter items separated by commas:
Chicken,Burger,Fries

Do you want to confirm your order? yes

---- BILL ----
Chicken: $5.99
Burger: $4.99
Fries: $2.99

Total: $13.97
```
